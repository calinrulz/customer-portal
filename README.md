# Welcome to Acme Bank Customer Portal
To get started follow the instructions

## Open Terminal

### First tab
* `cd portal-api`
* `npm install`
* `npm start`

### Second tab
* `cd portal-client`
* `npm install` / `yarn install`
* `npm start` / `yarn start`

### Go to `portal-api` or `portal-client` to find more info regarding the project
